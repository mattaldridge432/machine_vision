# machine_vision

-Automated litter event detection with video clip exports.

-Please contribute comments, feedback and improved versions so that we can solve the litter problem.



-kbtcam_v3.py uses a camera feed

-kbtwindow_v3m.py uses a capture region on the left hand side of the screen. To use the program, drag the display window to the right side of the screen and the webcam display window to the capture region on the left. There are infinity mirror style visual effects within the window while opening and moving windows into place, this is normal and expected.

-vision_system.py and vision_client.py work together (not yet developed or tested fully).

-vision_system_2.py and vision_client_2.py work together (not yet developed or tested fully).

-vehicle.py and vehicle_client.py work together (untested).
