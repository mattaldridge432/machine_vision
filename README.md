# machine_vision

-Machine Vision for automated litter event detection with litter event video clip exports.
-Please contribute comments, feedback and improved versions so that we can solve the litter problem.


-kbtcam uses a camera feed

-kbtwindow uses a capture region on the left hand side of the screen. To use the program, drag the display window to the right side of the screen and the webcam display window to the capture region on the left. There are infinity mirror style visual effects within the window while opening and moving windows into place, this is normal and expected.

-vision_system and vision_client work together (not yet developed or tested fully).

-vehicle and vehicle_client work together (untested).
